# 🌐 Portfolio Website (React + Tailwind)

Personal animated portfolio built with **React** and **Tailwind CSS**.

It’s designed to be **clean, responsive, and easy to customize** using data-driven sections (About, Skills, Experience, Work/Projects, Education, Contact).

## ✨ What’s included

- **Animated UI layer** (background grid + blur blob)
- **Navbar + section-based layout**
- **Skills** grouped by category with logo icons
- **Experience** timeline-style cards
- **Projects/Work** section with tags and links
- **Education** section
- **Contact** section and a shared **Footer**

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Vercel (recommended for deployment)

## 🧑‍💻 How to run

From inside `kithusi/`:

```bash
npm install
npm run dev
```

## 🏗️ Build for production

```bash
npm run build
npm run preview
```

## 🔧 Customization

Most content lives in `src/constants.js`.

- Update **skills**, **experience**, **education**, and **projects** arrays there
- Edit section components under `src/components/` if you want to change layout/styles

## 📌 Notes

This repository is a front-end portfolio project—no backend required for local development.
